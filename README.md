# AI-Powered Data Understanding & ETL System

## Overview
This project is an AI-driven system that automates data analysis, ETL recommendation, and data drift detection.

## Features
- Dataset profiling (missing values, duplicates)
- AI-based ETL recommendations
- PySpark-based ETL processing
- Data drift detection using statistical methods

## Tech Stack
- Python
- Streamlit
- PySpark
- Pandas
- SciPy

## How to Run
pip install -r requirements.txt
python -m streamlit run app.py

## Use Case
Useful in real-world ML pipelines to monitor data quality and prevent model degradation.
